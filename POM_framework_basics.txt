POM (Page Chaining Model)


1. Page Layer (Having Page Objects and methods/ Object Repository(Collection of all the web objects))
2. Base Class (Having common driver it is actually the parent class of all the classes inside Page Layer as well as Test Layer : Driver Init, properties, etc)
3. Test Layer (Having test cases wrt each page : uses TestNG annotations)
4. Config Layer (Having the config.properties file exist for setting up Environment variables (i.e., URL, uname,pwd,browser,etc)
5. Test Data Layer(conatining excel files with different sheets )
6. Util Layer (Excel util, SS util, other special methods for sending mail or other common util, Generic methods)
7. Report Layer (HTML, TestNG, ExtentReports etc)

Technical Stack:

1.Maven
2.Java
3.Apache POI
4.Extent/testNG 
5.Selenium
6.Log4j
7.Jenkins
